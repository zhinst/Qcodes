qcodes.InstrumentChannel¶

class qcodes.InstrumentChannel(parent: Union[qcodes.instrument.base.Instrument, InstrumentChannel], name: str, **kwargs)[source]¶

Base class for a channel in an instrument

Parameters:
  • parent (Instrument) – the instrument to which this channel should be attached
  • name (str) – the name of this channel
name¶

str – the name of this channel

Dict[Parameter] – All the parameters supported by this channel. Usually populated via add_parameter
