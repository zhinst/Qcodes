class qcodes.utils.zmq_helpers.Publisher(topic: str, interface_or_socket: str = 'tcp://localhost:5559', timeout: int = 10000, hwm: int = 20833330, context: zmq.sugar.context.Context = None)[source]¶

Bases: qcodes.utils.zmq_helpers.UnboundedPublisher

Publisher. Allows for a publisher that will not use all the memory. Tune the timeout and hwm to fit the needs of the situation. We start with very permissive defaults:

  • 10 seconds linger

  • 2.5 GB cache

NOTE that this offers no guarantees on message delivery. If there is no reciever the message is LOST.

Parameters

  • interface_or_socket – Interface or socket to connect to

  • topic – Topic of this publisher

  • timeout – time in millisecond to wait before destroying this published and the messages it caches

  • hwm – number of messages to keep in the cache

  • context – Context to reuse if desired
