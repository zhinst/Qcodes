Public¶

Classes and Functions¶

This page lists the entrypoints to the plubic qcodes API.

Config¶

Config(path) QCoDeS config system

Station¶

station.Station(*components, monitor, …) A representation of the entire physical setup.

Loops¶

Loop(sweep_values[, delay, station, …]) The entry point for creating measurement loops

Measure¶

measure.Measure(*actions) Create a DataSet from a single (non-looped) set of actions.

Actions¶

Task(func, *args, **kwargs) A predefined task to be executed within a measurement Loop.
Wait(delay) A simple class to tell a Loop to wait <delay> seconds.
BreakIf(condition) Loop action that breaks out of the loop if a condition is truthy.

Data¶

DataSet([location, arrays, formatter, io, …]) A container for one complete measurement loop.
new_data([location, loc_record, name, …]) Create a new DataSet.
load_data([location, formatter, io]) Load an existing DataSet.
FormatLocation([fmt, fmt_date, fmt_time, …]) This is the default DataSet Location provider.
DataArray([parameter, name, full_name, …]) A container for one parameter in a measurement loop.
Formatter Data file formatters
GNUPlotFormat([extension, terminator, …]) Saves data in one or more gnuplot-format files.
DiskIO(base_location) Simple IO object to wrap disk operations with a custom base location.

Instrument¶

Function(name[, instrument, call_cmd, args, …]) Defines a function that an instrument can execute.
Parameter(name, instrument, label, unit, …) A parameter that represents a single degree of freedom.
StandardParameter(name[, instrument, …])
ArrayParameter(name, shape, instrument, …) A gettable parameter that returns an array of values.
MultiParameter(name, names, shapes, …) A gettable parameter that returns multiple values with separate names, each of arbitrary shape.
ManualParameter(name[, instrument, …]) A simple alias for a parameter that does not have a set or a get function.
SweepFixedValues(parameter[, keys, start, …]) A fixed collection of parameter values to be iterated over during a sweep.
SweepValues(parameter, **kwargs) Base class for sweeping a parameter.
combine(*parameters, name[, label, unit, …]) Combine parameters into one sweepable parameter
CombinedParameter(parameters, name, label, …) A combined parameter
Instrument(name, metadata, VT]] = None, **kwargs) Base class for all QCodes instruments.
IPInstrument(name[, address, port, timeout, …]) Bare socket ethernet instrument implementation.
VisaInstrument(name[, address, timeout, …]) Base class for all instruments using visa connections.
InstrumentChannel(parent, …) Base class for a channel in an instrument
ChannelList(parent, name, chan_type, …) Container for channelized parameters that allows for sweeps over all channels, as well as addressing of individual channels.
